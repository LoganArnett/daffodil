# Macbook Component

## Purpose
This component serves a way to render an image inside the screen of a laptop.

## Aspect Ratio

|Display Size|Aspect Ratio|
|---|---|
|1280×800|16:10|
|1440×900|16:10|
|1680×1050|16:10|
|1920×1200|16:10|
|1366x768|16:9|
|1920x1090|16:9|

## Usage
```
<daffio-macbook>
    <img src="/path/to/some/file.jpg"/>
</daffio-macbook>
```
